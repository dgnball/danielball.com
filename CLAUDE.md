## What this repo is

This is the source code for [danielball.com](http://danielball.com), a personal portfolio and notes site built with
Jekyll and hosted on GitHub Pages.

## Writing style

When writing or editing content for this site, follow these rules.

**Voice and tone**

- Write in a clear, direct and conversational style
- Use first person where appropriate
- Sound like a knowledgeable professional but not a pompous one
- Keep sentences short and to the point
- British English spelling throughout

**Grammar rules**

- No Oxford commas
- No em-dashes or en-dashes in prose. Rewrite the sentence instead
- No colons or semicolons in prose. Use separate sentences
- Dashes are fine in lists and code examples
- Avoid overly formal constructions. "I worked on X" beats "I was responsible for the delivery of X"

**What to avoid**

- Buzzwords and filler phrases ("leverage", "utilise", "cutting-edge", "synergy")
- Passive voice where active voice is cleaner
- Padding sentences that don't add information
- Starting bullets with the same word repeatedly
